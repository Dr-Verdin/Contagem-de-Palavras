# Contagem-de-Palavras
Atividade de SCC0222 – Laboratório de Introdução à Ciência da Computação I

## Descrição

Implemente um programa em C que leia o nome de um arquivo de texto, e em seguida leia uma palavra, de no máximo 10 caracteres, que será buscada dentro do arquivo. O programa deve imprimir quantas vezes essa palavra dentro do texto do arquivo. Tenha em mente que a palavra pode aparecer maiúscula ou minúscula, e também seguida por pontuação, não conte situações em que a palavra aparece dentro de outra palavra.
